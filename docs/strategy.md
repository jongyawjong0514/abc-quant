# Strategy Design

## 初始研究方向

本專案初始方向是研究台股中短期主升段起漲點，使用：

- 價量結構。
- 趨勢與波動。
- 籌碼變化。
- 基本面動能。
- 市場狀態。

## 不直接追求的事項

初期不直接追求：

- 複雜深度學習模型。
- 高頻交易。
- 券商自動下單。
- 只針對單一股票最佳化。

## Baseline 策略

第一版 baseline 可包含：

1. 20 日動能排序。
2. 60 日動能排序。
3. 量增價漲條件。
4. 營收年增率排序。
5. 等權重持有 top N。

## 研究重點

策略研究不得只看最終報酬，必須同時分析：

- 交易次數。
- 換手率。
- 最大回撤。
- 空頭市場表現。
- 牛市與震盪市表現差異。
- 對交易成本敏感度。
